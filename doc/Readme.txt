===============================================================================

　HTT_RPC

===============================================================================


■ソフトの概要

　ウェブサーバーです。
　全てのリクエストは Remote Procedure Call のようにプログラムで応答します。
　対応している言語は C#, Java です。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　WHTTR.exe を実行して下さい。


■終了方法

　以下の何れかの方法で終了できます。

　　タスクトレイ上のアイコンを右クリックして「終了」を選択する。

　　WHTTR.exe /T を実行する。


■サーバーの仕組み

　以下のようにHTTPリクエスト・レスポンスを渡します。

　                      ネットワーク                    プロセス間通信           ユーザーが実装したプログラム
　+--------------+                        +-------+                        +-----------+--------------------------+
　|              |  -- HTTPリクエスト --> |       |  -- HTTPリクエスト --> |           |                          |
　| クライアント |                        | WHTTR |                        | HttServer | ユーザーが実装したコード |
　|              | <-- HTTPレスポンス --  |       | <-- HTTPレスポンス --  |           |                          |
　+--------------+                        +-------+                        +-----------+--------------------------+


■WHTTRの制限

　同じＩＰアドレスからの最大同時接続を２０に制限しています。


■取り扱い種別

　フリーソフト
