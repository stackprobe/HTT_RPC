===============================================================================

　HTT_RPC

===============================================================================


■ソフトの概要

　ウェブサーバーです。
　全てのリクエストは Remote Procedure Call のようにプログラムで応答します。
　対応している言語は C#, Java です。


■動作環境

　Windows 10


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　WHTTR.exe を実行して下さい。


■終了方法

　以下の何れかの方法で終了できます。

　　タスクトレイ上のアイコンを右クリックして「終了」を選択する。

　　WHTTR.exe /T を実行する。


■サーバーの仕組み

　以下のようにHTTPリクエスト・レスポンスを渡します。

　                      ネットワーク                    プロセス間通信           ユーザーが実装したプログラム
　+--------------+                        +-------+                        +-----------+--------------------------+
　|              |  -- HTTPリクエスト --> |       |  -- HTTPリクエスト --> |           |                          |
　| クライアント |                        | WHTTR |                        | HttServer | ユーザーが実装したコード |
　|              | <-- HTTPレスポンス --  |       | <-- HTTPレスポンス --  |           |                          |
　+--------------+                        +-------+                        +-----------+--------------------------+


■WHTTRの制限

　同じＩＰアドレスからの最大同時接続を２０に制限しています。

　- - -

　本サーバーは送受信データを一時ファイルに書き出します。
　仮に不正なリクエストがあった場合、不正な受信データを書き出した一時ファイルがウィルスと見なされ
　ウィルス対策ソフト等によって隔離等の処置が行われる可能性があります。

　一時ファイルの場所は、環境変数 TMP の指すフォルダの配下、ローカル名は以下のとおりです。

　　受信データ ... Recv.httdat
　　送信データ ... Send.httdat


■取り扱い種別

　フリーソフト
